Projects

Mushroommania

Mushroomania is a visualization project that enables the user to navigate and analyze mushroom data based on the data gathered by Wagner, D., Heider, D. & Hattab, G. using a natural language processor from the Mushrooms & Toadstools. Collins, 2012 book by Patrick Hardin.

The project was developed for the Data Visualization (IN4089) course at The Delft University of Technology.

Uno-Chess - Chess Variant

Uno-Chess is a modification of normal chess in which capture rules have been reversed. This means that your pieces can only be captured by enemy pieces that your piece would have under attack in normal chess. Additionally, you can not make normal captures, only enemy pieces which would target your piece in normal chess can be captured. In this gamemode pawns are more valuable than queens and the 8 surrounding squares of the king should be guarded closely.

Running Average Power Limit Measurement

The Running Average Power Limit Measurement Tool is a command line power consumption analysis tool that uses hardware counters on the AMD Ryzen platform and INTEL platform. It periodically reads the average power consumption per cpu package over a time interval and outputs them as csv values. Its secondary mode allows for the measurement of total energy consumption over a specific duration or until exit.

VSCode Power Measurement

The VSCode Power Measurement Extension offers a build in overview of power consumption of a specific process using a fully integrated graph into VSCode. It provides an overview of previous measurements with a short summary of the measurement including process name, start time, average power usage and maximum power usage. The extension offers persistent storage for measurements and quick comparison of previous and currents measurements by opening multiple graphs side by side.

The project was developed for the Sustainable Software Engineering (CS4415) course at The Delft University of Technology.

CT-Scan Visualization

The CT-Scan Visualization Software offers a fully functioning user interface, which allows for visualization using Ray Casting with 1D and 2D Transfer functions to create Compositional, Maximum Intensity, and Iso-surface Projections. Additionally, it offers the user the option to choose between Nearest Neighbour, Tri-Linear, and Tri-Cubic Interpolation and clearer visualization using fully customizable Phong and Tone Shading. All features are fully integrated and can be used interchangably.

The project was developed for the Data Visualization (IN4089) course at The Delft University of Technology.
